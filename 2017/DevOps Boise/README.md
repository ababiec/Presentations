## DevOps with Visual Studio Team Services
#### DevOps Boise | 16 June 2017
Microsoft’s Visual Studio Team Services is a complete DevOps solution that enables collaboration and automates software delivery for any language, targeting any platform. This session will primarily be a demonstration of using Visual Studio Team Services to plan, manage, and execute the application development process including storing code in a private Git repo, creating a build that runs automated unit tests, and creating a release that runs deployment tests.
